# asciimoticons Snippet Extension for VSCode

ascii emoticon snippets to improve your comments and code (ノಠ益ಠ)ノ彡┻━┻

## Supported Languages

The snippets are available in files of the following languages:

- css
- elixir
- html
- javascript / javascriptreact
- json
- plaintext
- python
- scss
